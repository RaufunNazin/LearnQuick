Frequently Asked Questions

Here are some of the frequent questions from you.
Grid can be used for two-dimensional layout where flexbox is used for one dimensional design, row or column. For designs with multi-row or multi-column layout, it is easier to use the grid model rather than flexbox.
Bootstrap is heavier than Tailwind. It focuses on building sites with pre-styled components where Tailwind focuses on fast Development using smaller and faster components. Also Tailwind is heavily customizable on various stuff where bootstrap is limited in customization.
The CSS Box Model shows us the proper positions of some properties such as Margin, Padding, Border etc. We can learn their respective position in a desgin by looking at the CSS Box Model because it tells us where the properties or attributes will be placed.
Semantic tags are tags which provide meaning for a div rather than just simply saying out "div". Few of these tags are header , footer and article tag which can carry a meaning by themselves being easily understandable of the contents inside those tags.
